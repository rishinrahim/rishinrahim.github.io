2021 - Present
As a Sr. Machine Learning engineer at Blueoptima,I currently lead the development of machine learning models capable of detecting AI generated code, assessing the impact of AI on developer productivity and quality. We are also working on building a model that can identify developer signatures within code, that significantly aids in authorship. We are also working on Fine-tuning state-of-the-art code generation LLM models such as Code Llama and Mistral to address code vulnerabilities. These models, fine-tuned on datasets featuring code vulnerabilities, significantly improved the accuracy of automated code repair techniques. Previously, I contributed to the development of models capable of discerning developer behaviour based on version control data and commit patterns. I also worked on the development of a model to map OSS dependency vulnerabilities to published vulnerabilities in NVD by analysing text data from JIRA Bug reports, Github issues, release notes, and security advisories. My role also extends to managing and maintaining the end-to-end deployment of these models into the production environment. Our development/production environment is set up on an on-premise cloud, integrated with CUDA GPUs, using Kubernetes, MLflow, and Docker.
